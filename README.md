# 🤖 Basic Chatbot using Python (CodeAlpha Internship)

This project is a simple rule-based chatbot developed using Python. It interacts with users by responding to predefined inputs like greetings, questions, and exit commands.

## 🎯 Objective
To build a basic chatbot that can simulate conversation using conditional logic and loops.

## 🚀 Features
- Responds to greetings like "hello", "hi"
- Answers simple questions like "how are you"
- Tells its name
- Provides help instructions
- Displays current time (extra feature)
- Exit command using "bye"
- Handles unknown inputs gracefully

## 🛠️ Technologies Used
- Python
- Conditional Statements (if-elif)
- Loops (while loop)
- Functions
- datetime module (for time feature)

## 📂 How It Works
1. The chatbot continuously takes input from the user
2. Matches input with predefined conditions
3. Responds accordingly
4. Ends conversation when user types "bye"
